# TronWallet
年轻人真正意义上的第一个波场chrome插件钱包~

##安装方法:
1. 访问 https://github.com/micah13433/tron-chrome-wallet 下载zip文件，解压到一个目录
2. 打开chrome浏览器--> 更多工具 --> 扩展程序，勾选开发者模式。
3. 点击左侧 加载已解压的扩展程序 按钮，选择步骤1的目录，点击确定即可。

##功能模块：
0. 线上线下环境随意切换
1. 账号注册，登录，登出
2. 余额，交易记录查看，资产转入转出
3. 资产冻结，解冻
4. 超级代表投票
5. token发行，购买
6. 交易所行情，
7. 币市快讯讯
